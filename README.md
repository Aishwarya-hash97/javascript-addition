



\## ➕ JavaScript Addition Program



A simple web-based application built with HTML and JavaScript that performs addition of two user-input values. The program includes input validation to ensure only numeric values are processed and provides real-time feedback on the webpage.



\### 🚀 Key Features



\* Adds two numbers entered by the user

\* Validates inputs to accept only numeric values

\* Displays error messages for invalid inputs

\* Dynamically updates results without page reload

\* Includes screenshots for both valid and invalid cases



\### 🛠️ Technologies Used



\* HTML

\* JavaScript

\* DOM Manipulation

\### ⚙️ How It Works



1\. Enter two values in the input fields

2\. Click the \*\*Add\*\* button

3\. The program validates the inputs

4\. If valid, the sum is calculated and displayed

5\. If invalid, an error message is shown



\### 🔍 Core Functionality



\* \*\*Input Validation\*\*: Uses `isNaN()` to check if inputs are numbers

\* \*\*Dynamic Calculation\*\*: Computes the sum using JavaScript

\* \*\*Error Handling\*\*: Displays messages for invalid inputs



\### 💻 Code Logic Overview



\* \*\*Validation Function\*\*: Ensures both inputs are numeric before proceeding

\* \*\*Addition Function\*\*: Calculates and displays the sum on the webpage



\### 💡 Purpose



This project is ideal for beginners learning JavaScript basics, especially input validation, DOM manipulation, and handling user interactions in web applications.
